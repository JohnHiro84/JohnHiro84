Webpage Shortening Service

This is a project from freecodecamp.

Microservice - Url Shortener

Enter a website in the form below with the format "www.google.com" to get a shortened version of the website


http://localhost:3000/UrlShortener

The main entry point for the web shortening service

http://localhost:3000/findAllUrls

This entry point will show all the different short link Urls created

http://localhost:3000/findAllUrlsPretty/

This entry point will show all the different short link Urls returned with an Ejs Template Engine

http://localhost:3000/shortUrl/64

The shortened link will be in the above format.

If you type this into your web browser you will be redirected to your webpage

http://localhost:3000/deleteAllUrls/

This entry point will delete all the weblinks created
