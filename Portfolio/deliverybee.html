Delivery Bee

A Postmates Clone

MERN Stack, Redux, MapBox and Stripe APIs

The Delivery Bee is a Postmates clone where customers can request deliveries, and Helper Bees fulfill the orders. The app has a frontend for the customers and a frontend for the helpers. Delivery Bee utilizes the MapBox API to display maps and the Stripe API to take care of payments. The app uses Redux to manage state, and it also uses React Hooks, toast notifications, ChartJs, and MomentJs(format time/dates).
