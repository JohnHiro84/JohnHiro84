City Events

A Meetup.com clone

Made with the MERN stack - MongoDB, Express, React, Node

This project uses mongoDB and Express connect to the back end that serves up JSON responses. It uses JavaScript Web Tokens as part of the authentication strategy. The front end is managed by React and uses Redux to manage state. Users can create new events to share and can leave comments and confirm attendance.
