Lets Cook

A recipe sharing application

Made with React/Redux, Rails, Node

This project uses rails to connect to the back end that serves up JSON responses. The front end is managed by React and uses Redux to manage state. Users can create new recipes to share with other users. Users can leave comments, tags, likes and edit thier own recipes.
