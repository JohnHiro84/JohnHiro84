Codeacademy features a pro intensive course called Build Front End Web Apps:

This course involved using javascript, node and ReactJS.

There was also a slack channel to work with fellow students.

The 3 main projects that we worked in the course:

  • Minesweeper
  • Ravenous
  • Jammming

Minesweeper:

This project features

  • Using nodeJs
  • Basic JS fundamentals

Ravenous

This project features

  • Using React and Components
  • Making API calls to the Yelp API

Jammming

This project was similar to the Ravenous Project, yet it was encouraged to try to do as much as you could on your own.

  • Using React and Components
  • Making API calls to the Spotify API
